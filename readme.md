# Docs

Documentation for Leada

## Setup

    bundle install
    rake

Go to localhost:4002 to see the site.

## Deploying

First time only:

    mkdir _deploy
    cd _deploy
    git remote add origin git@github.com:teamleada/docs.git
    cd ..

Then, you only need to run:

    rake deploy
